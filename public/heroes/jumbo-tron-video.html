Jumbo Tron 4 - Video Background

This is an example of a full width HTML 5 video background with content overlay and a pause button.

HTML:
                    
                    <div class="jumbotron jumbo-bgimage-4">
                    <video autoplay muted loop id="video-background">
                    <source src="https://media.w3.org/2010/05/sintel/trailer.mp4" type="video/mp4">
                    Your browser does not support HTML5 video.
                    </video>
                    
                    <div class="video-content-over">
                    <h1>Jumbo Tron 4 - Video Background</h1>
                    <p">This is an example of a full width HTML 5 video background with content overlay and a pause button.</p>
                    <button id="pause-button" onclick="myFunction()">Pause</button>
                    </div>
CSS:
                    
                    #video-background{
                        position: fixed;
                        right: 0;
                        bottom: 0;
                        min-width: 100%; 
                        min-height: 100%;
                    }
                    .video-content-over{
                        position: relative;
                        margin-top:10%;
                        background: rgba(0, 0, 0, 0.3);
                        color: #f1f1f1;
                        width: 100%;
                        padding: 20px;
                    }
                    #pause-button {
                        width: 200px;
                        font-size: 18px;
                        padding: 10px;
                        border: none;
                        cursor: pointer;
                    }
JS:
                    
                    var video = document.getElementById("video-background");
                    var btn = document.getElementById("pause-button");
                    
                    function myFunction() {
                        if (video.paused) {
                            video.play();
                            btn.innerHTML = "Pause";
                            btn.className -= " btn-success";
                            btn.className += " btn-danger";
                        } else {
                            video.pause();
                            btn.innerHTML = "Play";
                            btn.className -= " btn-danger";
                            btn.className += " btn-success";
                        }
                    }
